# 🎮 Intelligence Dashboard + IA

Este proyecto es un dashboard interactivo de análisis de datos de la industria de los videojuegos, potenciado por Inteligencia Artificial para generar insights automáticos.

## 🚀 Características
- **Análisis Global:** Visualización de ventas por región (NA, EU, JP) y globales.
- **Consultoría IA:** Integración con **Llama 3 (vía Groq)** para responder preguntas complejas sobre los datos en lenguaje natural.
- **Filtros Dinámicos:** Segmentación por editorial, género y plataforma en tiempo real.
- **KPIs en tiempo real:** Métricas clave calculadas automáticamente sobre el dataset.

## 🛠️ Tecnologías
- **Lenguaje:** Python 3.x
- **Librerías:** Streamlit, Pandas, Plotly
- **IA:** Groq API (Llama 3 70B)
- **Dataset:** Histórico de ventas de videojuegos (Excel)

## 📦 Instalación y Uso
1. Clona el repositorio.
2. Instala las dependencias: `pip install`
3. Ejecuta la app: `streamlit run dashboardia.py`
